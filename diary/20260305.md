# GitHub Pages でサイトを始めた

今日からテキスト日記サイトを始めることにした。

GitHub Pages を使えば、リポジトリにファイルをプッシュするだけで公開できる。サーバーの管理も不要で、しかも無料だ。シンプルな HTML と CSS だけで作れるので、余計なことを考えずに文章を書くことに集中できそうだ。

日記のスタイルについては、できるだけシンプルに保ちたい。フォントはセリフ体にして、背景は白に近い薄いクリーム色にした。読み返すときに目が疲れないようにしたかった。

これからどんなことを書いていくか、まだはっきりとは決めていないが、日々の気づきや出来事を素直に記録していければと思っている。
